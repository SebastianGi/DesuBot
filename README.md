## Getting Started

### Prerequisites

* discord.py - *I'm using version v0.16.12*
* urllib3
* [An osu!-API key](https://github.com/ppy/osu-api/wiki)

### Installation

Rename `config.default.json` to `config.json` and add your Data before the first use.
